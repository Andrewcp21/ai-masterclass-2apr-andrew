Jangan Naif

Tantangan Nyata yang Perlu Kamu Pertimbangkan

Challenge 01
Data Sensitif
Data karyawan, revenue, pelanggan — tidak boleh sembarangan dikirim ke API eksternal
  • Anonymize data sebelum diproses (nama → ID anonim)
  • Claude Code berjalan lokal — data tidak otomatis ke server
  • Untuk data sangat sensitif: pertimbangkan local model (Ollama)
  • Buat data classification policy di tim
Challenge 02
Biaya Langganan
Claude Code membutuhkan langganan berbayar untuk penggunaan penuh
  • Claude.ai Pro: ~$20/bulan ≈ Rp 330rb — sudah cukup untuk use case harian
  • Bandingkan dengan nilai 150 jam/tahun yang dihemat
  • Break-even: sekitar 2–3 minggu pemakaian pertama
  • Tulis instruksi yang spesifik — kurangi iterasi yang boros
Lanjutan

Dua Tantangan Lagi yang Sering Dilupakan

Challenge 03
Kualitas Output
Claude bisa salah interpretasi data atau memberikan angka yang tidak akurat
  • Selalu validasi angka penting secara manual
  • Minta Claude tampilkan data yang dia pakai untuk setiap klaim
  • Gunakan output AI sebagai draft, bukan final tanpa review
  • Semakin spesifik prompt → semakin akurat output
Challenge 04
Maintenance
Script yang tidak terdokumentasi jadi beban besar 3 bulan kemudian
  • Simpan instruksi/prompt sebagai file teks yang bisa diulang
  • Simpan di Git dari hari pertama
  • Buat README singkat: untuk apa, cara jalankan, siapa yang maintain
  • Test dengan data dummy sebelum pakai data production

Tetap Manual

Mungkin masih bisa bertahan 1–2 tahun lagi.

  • 4 jam bikin laporan yang sama setiap bulan

Apakah perlu bisa coding?

Claude.ai Pro ~$20/bulan (≈ Rp 330rb) sudah termasuk akses Claude Code. Untuk laporan mingguan, Pro lebih dari cukup.
